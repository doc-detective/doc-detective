## Fields

Field | Type | Description | Default
:-- | :-- | :-- | :--
tag | one of:<br/>- string<br/>- array of string | Optional. Element/component name(s) to match. | 
attributes | object | Optional. Map of attribute name to matcher: an exact/regex string, or `true` to require that the attribute exists. For Markdown/MDX, attributes come from Kramdown/Pandoc-style attribute lists (`{: .cls #id key="val"}`): classes normalize to `class`, IDs to `id`. | 
contentMatches | string | Optional. Regular expression the element's inner content must match. Capture groups are exposed as `match.<n>` capture paths.<br/><br/>Minimum length: 1 | 
contentExcludes | string | Optional. Regular expression the element's inner content must not match.<br/><br/>Minimum length: 1 | 
precededBy | string | Optional. Regular expression matched against the sibling text immediately before the node, for verb-context patterns (e.g. "Click **X**").<br/><br/>Minimum length: 1 | 
followedBy | one of:<br/>- string<br/>- object | Optional. Regular expression matched against the sibling text immediately after the node, or an object chaining a second node for sequence patterns. Chained-node fields are exposed as `then.<field>` capture paths. |
