## Fields

Field | Type | Description | Default
:-- | :-- | :-- | :--
language | one of:<br/>- string<br/>- array of string | Optional. Info-string language(s) to match. An empty string matches bare fences with no language. | —
metaMatches | string | Optional. Regular expression the info-string tail (after the language) must match.<br/><br/>Minimum length: 1 | —
metaExcludes | string | Optional. Regular expression the info-string tail must not match.<br/><br/>Minimum length: 1 | —
attributes | object | Optional. Map of attribute name to matcher: an exact/regex string, or `true` to require that the attribute exists. For Markdown/MDX, attributes come from Kramdown/Pandoc-style attribute lists (`{: .cls #id key="val"}`): classes normalize to `class`, IDs to `id`. | —
contentMatches | string | Optional. Regular expression the block content must match. Capture groups are exposed as `match.<n>` capture paths.<br/><br/>Minimum length: 1 | —
contentExcludes | string | Optional. Regular expression the block content must not match.<br/><br/>Minimum length: 1 | —
