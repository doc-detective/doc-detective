## Fields

Field | Type | Description | Default
:-- | :-- | :-- | :--
annotate | object([annotate](/reference/schemas/annotate)) | Required. Adds, updates, or clears annotations that stay on screen until something clears them. Unlike a screenshot's own `annotations` — which live only for that capture — these are drawn into the page, so they appear in recordings and in any screenshot taken while they're up. Annotations added here persist across steps and across navigation until an `annotate` step clears them, their `duration` elapses, or the context ends. | —
