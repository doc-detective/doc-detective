## Fields

Field | Type | Description | Default
:-- | :-- | :-- | :--
color | string | Optional. Foreground color — strokes, arrowheads, and text (hex, rgb, or named color). | —
background | string | Optional. Background color for text-bearing annotations (hex, rgb, or named color). Use `transparent` for none. | —
strokeWidth | number | Optional. Line width in pixels.<br/><br/>Minimum: 0 | —
fontSize | number | Optional. Font size in pixels.<br/><br/>Minimum: 1 | —
fontFamily | string | Optional. Font family. Falls back through the list as in CSS. | —
opacity | number | Optional. Opacity, from 0 (invisible) to 1 (opaque).<br/><br/>Minimum: 0. Maximum: 1 | —
radius | number | Optional. Corner radius in pixels, for boxes and text backgrounds.<br/><br/>Minimum: 0 | —
padding | number | Optional. Padding in pixels inside text boxes, and between an outline and its element.<br/><br/>Minimum: 0 | —
maxWidth | number | Optional. Maximum width in pixels for text before it wraps.<br/><br/>Minimum: 1 | —
intensity | number | Optional. Blur strength. Higher values obscure more.<br/><br/>Minimum: 1 | —
