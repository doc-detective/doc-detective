Field | Type | Description | Default
:-- | :-- | :-- | :--
type | string | Required. The type of integration. Currently supported: 'heretto'. Additional types may be added in the future.<br/><br/>Accepted values: `heretto` | 
integrationName | string | Required. The name of the integration configuration in the config file. Used to look up authentication credentials. | 
fileId | string | Optional. The unique identifier (UUID) of the file in the source CMS. If not provided, the file will be looked up by path. |
