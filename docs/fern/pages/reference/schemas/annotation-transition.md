## Fields

Field | Type | Description | Default
:-- | :-- | :-- | :--
enter | string | Optional. How the annotation appears. Use `none` for annotations that must never reveal what they cover — a `blur` that fades in shows the sensitive content underneath while it does.<br/><br/>Accepted values: `none`, `fade`, `pop`, `draw` | 
exit | string | Optional. How the annotation disappears.<br/><br/>Accepted values: `none`, `fade` | 
durationMs | number | Optional. Length of the enter and exit animations, in milliseconds.<br/><br/>Minimum: 0 |
