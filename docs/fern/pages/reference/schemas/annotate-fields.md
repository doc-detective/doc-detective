## Fields

Field | Type | Description | Default
:-- | :-- | :-- | :--
add | array of unknown | Optional. Annotations to draw. Give an annotation an `id` to update or clear it later. | 
update | array of unknown | Optional. Changes to annotations that are already on screen, matched by `id`. Each entry replaces the annotation with that id. | 
clear | one of:<br/>- boolean<br/>- array of string | Optional. Annotations to remove. `true` clears every annotation; a list of ids clears just those. |
