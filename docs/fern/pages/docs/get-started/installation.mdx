Downloading and running Doc Detective is straightforward. You can install Doc Detective using NPX (Node.js) or Docker. Follow the instructions for your preferred method:

Choose *NPX* if you have Node.js installed and want to install Doc Detective on your machine locally. Choose *Docker* if you can't install applications locally, prefer containerized environments, need to avoid Node.js installation, or want to isolate Doc Detective from your system.

Congratulations! You've run your first test with Doc Detective. :tada:

You should see output in the terminal indicating the progress and results of the test. Results are also saved in a `testResults-<timestamp>.json` file in the current directory (configurable with the `--output` option), and this test saves a screenshot to *example.png*.

## Lazy-install runtime

When you install Doc Detective via npm, the postinstall step downloads browser engines (Chrome, Firefox), drivers (ChromeDriver, Geckodriver), ffmpeg, and the npm packages that drive them. This keeps a fresh install ready to run tests immediately.

To skip this initial download and keep the install small, set the `DOC_DETECTIVE_AUTOINSTALL` environment variable before installing:

```bash
DOC_DETECTIVE_AUTOINSTALL=0 npm i -g doc-detective
```

Accepted values: `0`, `false`, `no`, or `off` (case-insensitive).

With this setting, runtime assets install lazily the first time a test needs them. You can also install them manually with `doc-detective install all`.

### Manual installation commands

Install specific runtime components:

```bash
# Install everything
doc-detective install all --yes

# Install only specific components
doc-detective install browsers chrome
doc-detective install runtime webdriverio appium
doc-detective install agents

# Check installation status
doc-detective install status
```

### Custom cache location

Override where Doc Detective stores runtime assets:

```bash
DOC_DETECTIVE_CACHE_DIR=/opt/doc-detective doc-detective install all --yes
```

This is useful in containers and CI environments.
