title: dragAndDrop
description: Drag an element to a target location.

The `dragAndDrop` action allows you to drag an element and drop it at a target location. This is useful for testing drag-and-drop interfaces, reordering lists, moving items between containers, and other interactive UI elements.

The `dragAndDrop` action requires both a source and target element, which can be specified using element text, CSS/XPath selectors, or detailed objects with additional configuration options.

To move between **points** instead of elements—carousels, sliders, plain scrolling—use [`swipe`](swipe), the movement subset of this action: `dragAndDrop` locates two elements and moves between them, while `swipe` moves between two coordinates (or a direction shorthand).

## Use cases

The `dragAndDrop` action is particularly useful for testing:

- **Kanban boards**: Moving tasks between different status columns
- **File uploads**: Dragging files from desktop to upload areas
- **List reordering**: Changing the order of items in sortable lists
- **Dashboard widgets**: Rearranging components on customizable dashboards
- **Form builders**: Moving form elements between different sections
- **Shopping carts**: Adding items by dragging them to the cart area

## Properties

The `dragAndDrop` action requires an object with the following properties:

- `source`: (Required) The element to drag from. Can be a string (element text or selector) or a detailed object.
- `target`: (Required) The element to drop onto. Can be a string (element text or selector) or a detailed object.
- `surface`: (Optional) The browser (session, window, or tab) the source and target elements live in, for example `{ "browser": "chrome", "tab": "board" }`. Omit to act on the active tab. See [Test across multiple tabs, windows, and browsers](/docs/test-docs/multiple-tabs-and-windows).

### Source and target element properties

When using detailed objects for `source` or `target`, you can specify:

- `elementText`: (Optional) Display text of the element. If combined with `selector`, the element must match both the text and the selector.
- `selector`: (Optional) [CSS selector](/docs/selectors/css) or [XPath selector](/docs/selectors/xpath) of the element. If combined with `elementText`, the element must match both the text and the selector.
- `timeout`: (Optional) Maximum duration in milliseconds to wait for this specific element to exist.

**Outputs:** the action exposes `outputs.sourceFound` and `outputs.targetFound`, booleans indicating whether the source and target elements were located. To capture them for later steps, use the step-level `variables` object with the expressions `$$sourceFound` and `$$targetFound`.

> For comprehensive options, see the full [`dragAndDrop`](/reference/schemas/draganddrop) reference.

## Examples

Here are several ways you might use the `dragAndDrop` action:

### Drag and drop using selectors or element text

{/* test {"testId":"dragdrop-selector-text","detectSteps":false} */}

```json
{
  "tests": [
    {
      "steps": [
        {
          "description": "Drag 'Item 1' and drop it onto 'Drop Zone'",
          "dragAndDrop": {
            "source": "#source-list .draggable-item:first-child",
            "target": "Drop Zone"
          }
        }
      ]
    }
  ]
}
```

{/* step {"stepId":"goto-dragdrop-text","description":"Open the drag and drop test page","goTo":"http://localhost:8092/drag-drop-test.html"} */}
{/* step {"stepId":"dragdrop-text","description":"Drag a widget by selector and drop it onto a zone by text","dragAndDrop":{"source":".draggable[data-widget='text']","target":"Drop Zone 1"}} */}
{/* test end */}

### Drag and drop with detailed element objects

{/* test {"testId":"dragdrop-detailed-objects","detectSteps":false} */}

```json
{
  "tests": [
    {
      "steps": [
        {
          "description": "Drag a specific task to a different column",
          "dragAndDrop": {
            "source": {
              "selector": ".task-card",
              "elementText": "Complete documentation"
            },
            "target": {
              "selector": ".column",
              "elementText": "In Progress"
            }
          }
        }
      ]
    }
  ]
}
```

{/* step {"stepId":"goto-dragdrop-object","description":"Open the drag and drop test page","goTo":"http://localhost:8092/drag-drop-test.html"} */}
{/* step {"stepId":"dragdrop-object","description":"Drag and drop using detailed source and target objects","dragAndDrop":{"source":{"selector":".draggable","elementText":"Widget Item 1"},"target":{"selector":".drop-zone","elementText":"Drop Zone 2"}}} */}
{/* test end */}
